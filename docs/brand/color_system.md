# Flocken - Färgschema & Design System

## Färgpalett

### Huvudfärger
```css
--primary: #6B7A3A;        /* Olivgrön - knappar, headers, aktiva element */
--secondary: #A29D89;      /* Ljusgrå/beige - sekundär text och ikoner */
--accent: #8BA45D;         /* Mjuk grön - hover states och interaktioner */
```

### Bakgrunder
```css
--background: #FFFFFF;     /* Vit - huvudbakgrund */
--card: #E8DCC0;          /* Sand/ljusbeige - kort, listor, popups */
--card-alt: #FFFFFF;      /* Alternativ kortbakgrund */
--card-dark: #D4C4A8;     /* Mörkare sand */
```

### Text
```css
--text-primary: #3E3B32;   /* Mörkbrungrå - rubriker */
--text-secondary: #A29D89; /* Ljusgrå/beige - sekundär text */
--text-light: #FFFFFF;     /* Vit text på mörk bakgrund */
```

### Status
```css
--error: #C44536;          /* Dämpad röd */
--success: #6B7A3A;        /* Olivgrön */
--warning: #D4A574;        /* Varm orange/sand */
```

### Specialfärger
```css
--male: #5A6631;           /* Mörkare olivgrön - hanar */
--female: #8BA45D;         /* Mjuk grön - tikar */
--favorite: #C44536;       /* Dämpad röd - hjärta/favorit */
```

## Användning

### Knappar
- **Primary CTA:** `#6B7A3A` (olivgrön)
- **Hover:** `#8BA45D` (mjuk grön)
- **Text på knappar:** `#FFFFFF` (vit)

### Kort & Sektioner
- **Alternating sections:** Vit (`#FFFFFF`) och sand (`#E8DCC0`)
- **Elevated cards:** Sand (`#E8DCC0`) med subtle shadow

### Typografi
- **Headlines:** `#3E3B32` (mörkbrungrå)
- **Body text:** `#3E3B32` med 0.9 opacity för läsbarhet
- **Secondary text:** `#A29D89`

## Tonalitet
Färgschemat ger en **naturlig, varm och jordnära** känsla som passar Flockens "vuxna, varma, vardagsnära" tonalitet.
